## 🧭 &nbsp; About me

I started out as a full-stack engineer who liked owning the whole flow — backend APIs, frontend interfaces, databases, deployments, and everything in between. Over time this pulled me deeper into **cloud systems, distributed services, and the kind of architecture work that keeps real products running**.

That foundation still drives how I work: *design the system first, understand the data, make it scale, then build the parts on top of it.*

## 🔬 &nbsp; What I'm exploring now

> **React Native + modern mobile patterns** — building mobile apps that feel smooth, responsive, and actually useful day-to-day. Good UI, clean navigation, offline behavior, talking to smart backends.

> **Multi-agent AI systems** — not just calling an LLM. Coordinating multiple reasoning units that can research, review, summarize, plan, or decide together. Agent orchestration, contextual memory, retrieval.

> **Where mobile meets AI** — the direction I'm most excited about. Apps that respond to the world, remember context, and behave more like partners than interfaces.
